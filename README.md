# Aplikasi FutsalConnect

**FutsalConnect** adalah platform manajemen reservasi lapangan futsal berbasis web yang dirancang untuk memudahkan pemain dalam melakukan *booking* lapangan dan mempermudah admin dalam mengelola operasional lapangan secara *real-time*.

## 🚀 Fitur Utama
* **Sistem Reservasi Real-Time:** Pengecekan slot waktu yang akurat (otomatis memblokir jadwal yang sudah dipesan/dikonfirmasi).
* **Dasbor Admin:** Pusat komando operasional untuk memantau pendapatan, status pesanan, serta statistik popularitas lapangan melalui visualisasi grafik.
* **Sistem Keputusan Wasit:** Fitur bagi admin untuk mengelola status pesanan (Approved/Rejected) secara langsung.
* **User Authentication:** Sistem login dan registrasi yang aman untuk setiap pemain.
* **Responsif UI:** Tampilan yang dioptimalkan untuk perangkat *desktop* maupun *mobile* dengan tema *Dark Mode* yang modern.

## 🛠 Tech Stack
* **Frontend:** HTML5, CSS3 (Custom Design), Bootstrap 5, Chart.js.
* **Backend:** PHP Native (terstruktur).
* **Database:** MySQL.
* **Hosting:** InfinityFree.

## 📦 Cara Menjalankan
1. *Clone* repositori ini ke dalam folder `htdocs` XAMPP Anda.
2. Impor file database `.sql` yang telah disediakan ke dalam phpMyAdmin.
3. Sesuaikan konfigurasi database pada file `koneksi.php`.
4. Buka aplikasi melalui peramban dengan mengakses `localhost/nama-folder-proyek`.

*Catatan: Untuk file database (.sql), silakan hubungi tim pengembang karena file database tidak disertakan di repositori publik demi keamanan.*
